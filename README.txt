Langkah untuk memakai aplikasi
- Deploy .war
- Import databasenya
- Masuk ke home page
    - Ada dropdown menu untuk memilih tipe masukan berita yang akan diklasifikasi
    - Jika memilih text, maka pastekan artikel berita ke text area
    - Jika memilih csv, browse file csv yang akan diklasifikasi
    - Jika memilih link, Pastekan link berita di textbox yang tersedia
    - Jika memilih crawler, pastekan link awal yang akan di crawl di textbox yang tersedia

Setelah berita diklasifikasi, pengguna dapat melakukan koreksi dengan mengklik tombol koreksi, kemudian memilih kategori yang benar, dan klik OK untuk dibuat model lagi.

Anda dapat mengebuild ulang data training pada homepage yang dieksekusi setelah user memasukkan masukan kategori yang benar
